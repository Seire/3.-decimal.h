Проект представляет собой создание нового типа данных в С. Работает аналогично Decimal из C#. Созданы необходимые арифметические операторы и функции для него (умножение, деление, вычитание, сложение, операторы сравнения и функции для работы с decimal типа обрезания десятичной части, смены знака, округления)
